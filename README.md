Midad

Midad is a modern note-taking application built with the MERN Stack (MongoDB, Express.js, React, and Node.js).
In Midad, every note is called a Memo, providing users with a simple and organized way to capture ideas, tasks, and important information.

✨ Features
Create, edit, and delete memos
Responsive and user-friendly interface
Secure user authentication and authorization
Persistent data storage with MongoDB
Real-time validation and error handling
Fast and scalable REST API
Search and organize memos efficiently
